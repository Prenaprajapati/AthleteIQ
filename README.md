# AthleteIQ 🏃

AthleteIQ is a web application designed to help athletes track their performance, training sessions, personal bests, nutrition, and recovery.

## Features

### User Authentication
- Login Page
- Registration Page

### Dashboard
- Training Statistics
- Weekly Progress Chart
- Personal Best Records
- Recent Workouts

### Planned Features
- Training Log
- Nutrition Tracker
- Injury Tracker
- Performance Analytics
- Goal Setting
- Athlete Profile Management

## Tech Stack

- HTML
- CSS
- Python (Planned)
- Flask (Planned)
- MySQL (Planned)
